Quadrix


HELLO! Welcome to"Quadrix".This is a two-player gaming interface built for playing [4*4] matrix game.This was specially designed for timepass so please consider it and hope that you might like this game.Support us

Key points to reference :-
  1. First enter your name on the player name bars.
  2. Click on start button to start the game.
  3. Devil = dark mode || Angle = light mode || Change BG = change the color of the background.
  4. Score will be displayed on the respective score boards
  5. Game always starts from Y turn.Select the Y player yourself.
  6. Hope you might like this game.
  7. Game win matrix:
    • Horizontal row (?)
    • vertical col (?)
    • Cross section(?)
    • Square in edges
    • Square in center
  8. THANK YOU AND PLAY WELL
