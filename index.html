HOW TO PLAY

  1. Enter player names in the designated fields
  2. Click START to begin the game
  3. First player uses Y, second player uses O
  4. Choose game modes for different visual experiences
  5. Scores are tracked on the scoreboard
WINNING PATTERNS
  • Horizontal row (4 in a row)
  • Vertical column (4 in a column)
  • Diagonal (4 in a diagonal)
  • Square in edges or center
